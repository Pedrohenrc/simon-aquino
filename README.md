# 📚 Online Bookstore

Sistema de livraria online desenvolvido para permitir a navegação, busca e compra de livros de forma simples e intuitiva.

A aplicação oferece uma experiência completa de e-commerce básico, permitindo que usuários explorem o catálogo, adicionem itens ao carrinho e finalizem pedidos.

## 📌 Visão Geral

A Online Bookstore é uma aplicação web focada na venda de livros, com interface amigável e funcionalidades essenciais para um fluxo de compra eficiente.

O sistema foi construído utilizando tecnologias web modernas no frontend e Python no backend, garantindo desempenho, simplicidade e facilidade de manutenção.

## 🚀 Funcionalidades

- 📖 Listagem de livros disponíveis
- 🔎 Busca por título, autor ou categoria
- 🛒 Carrinho de compras
- ➕ Adição e remoção de livros do carrinho
- 💰 Cálculo automático do total da compra
- 📦 Finalização de pedidos
- 📚 Visualização de detalhes do livro
- 🗂️ Organização por categorias
